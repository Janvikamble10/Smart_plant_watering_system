# Firmware

This folder contains the embedded firmware for the Smart Plant Watering System.

## Main File
- `irrigation_system.ino`

## Functionality
- Reads soil moisture using the Arduino ADC
- Applies threshold-based decision logic with hysteresis
- Controls a relay to turn the water pump ON or OFF

## Notes
- Firmware is written for Arduino UNO
- Designed and tested in simulation (Tinkercad)
