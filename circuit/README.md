# Circuit

This folder contains the circuit schematic and simulation views for the Smart Plant Watering System.

## Contents
- `tinkercad_circuit.png` – Simulation wiring view
- `tinkercad_schematic.pdf` – Schematic representation

## Notes
- Circuit was created and tested using Tinkercad Circuits
- Relay is used to isolate the pump power from the microcontroller
- Pump remains OFF by default using the normally-open (NO) relay contact
